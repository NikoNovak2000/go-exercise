# Go Course - The Complete Guide

This is a git repository for learning Go.

## Description

Following a [Udemy Course](https://www.udemy.com/course/go-the-complete-guide/) about Go from the ground up and in depth by building multiple demo projects and through exercises.

## Getting Started

### Requirements

- IDE Editor, Visual Studio Code - [installation for VSC](https://code.visualstudio.com/)
- Go - [installation for Go](https://go.dev/doc/install)

### Executing program

- Run the programs in terminal with command

```
 go run .
```
